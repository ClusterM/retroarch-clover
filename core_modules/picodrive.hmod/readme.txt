=== PicoDrive core for RetroArch ===

This module will add support for Sega Master System, Mega Drive / Genesis, Game Gear and Sega 32X games to RetroArch

Available executables and arguments:
- /bin/32x <rom> <clover_args>
  runs "picodrive" core
